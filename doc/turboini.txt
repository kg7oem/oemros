



Types
  * scalar
  * table (key/value pairs)
  * list

Scalar formats
  * string
  * boolean (true, false)
  * integer (1, 2, 3, -1)
  * real (1, 2, -1.1, inf, -inf, nan)
  * imaginary (1i, 2i, -1i, 1.1i)
  * complex (1+1i, 2-i)



Behavior
  * Multiple sections with the same name is fine (maybe configurable)
  * values can exist outside of any section
